Project Setup (VDC AutoCAD Plugin)

As part of my work on the AutoCAD Plugin, I was assigned the automation of an existing Project Setup process, which the Virtual Design and Construction team uses to set up the project directory for a newly contracted job.

The process consists of 3 phases; 1) Setting up folders and drawing for each trade/department per level of the building, 2) Setting up tabs representing a different view as well as inserting external references, and finally 3) Standalone functions that allow modifications to the results achieved from Phases 1 & 2.

This increased my exposure to AutoCAD - both the .NET API and Core Console (command line). I designed and implemented both UI and code for all 3 phases, and in collaboration with the rest of my team created a single Excel Spreadsheet (File Processing sheet) to hold all data that the VB.NET application would need to run all processes (Project Setup, Posting and Updating).

For Project Setup Phase 1, the program uses the File Processing Sheet to generate dummy drawing files for each level per trade, and inserts the respective trade reference tag into each drawing file created. I modified the process to allow the user to choose default insert location (0,0,0) or a custom location for reference tags being inserted.

For Project Setup Phase 2, a master template drawing is used to determine how many layout tabs need to be created for only the trades selected by the user. Using the AutoCAD .NET API, we are able to gather this information from the master template file and create the needed tabs, modify the necessary information, and finally insert the required external references. This process has gone from being a full day's work to 30 minutes or less in the worst-case scenario (setup for all trades with 20 or more levels).

Phase 3 allows users to perform the following; 1) Regenerate data retrieved from master template drawing, 2) update layout tab information, 3) Relocate trade reference tags for selected trades, and 4) Re-import external references for selected trades.

Category: .NET (VB, C#) // Visual Studio // AutoCAD // Microsoft Excel // Plugin Development
